# Meshtastic Guides

Questa è la “base operativa” per guide Meshtastic: app, firmware, hardware, MQTT, mappe, best practice e troubleshooting.

## Come è pensato il progetto
- **Una funzione = una pagina** (quando possibile)
- Android e iOS hanno percorsi UI diversi → li separiamo senza confusione
- Ogni procedura deve includere **verifica** (“come capisco che funziona davvero?”)

## Parti da qui
- **Come usare queste guide** (formato e convenzioni)
- **Fondamenta** (concetti comuni)
- Poi scegli: **App**, **Firmware**, **Hardware**, **MQTT**, **Mappe**
