# Come usare queste guide

Queste guide sono organizzate per **funzioni** (non per “feature buttate a caso”), perché Meshtastic evolve e le UI cambiano.

## Struttura standard di una pagina
1. **A cosa serve**
2. **Quando usarla / quando NON usarla**
3. **Prerequisiti**
4. **Percorso UI**
5. **Procedura passo‑passo**
6. **Verifica**
7. **Problemi comuni (con fix)**
8. **Note avanzate** (batteria, rete, sicurezza)

## Convenzioni
- Percorsi UI in monospace: `Settings > ...`
- Config in blocchi codice
- Warning importanti come “admonition”

!!! warning "Sicurezza"
    QR di canale, preset LoRa e configurazioni condivise possono cambiare comportamento del nodo.
    Prima di applicare configurazioni “di altri”, leggi sempre cosa stai importando.
