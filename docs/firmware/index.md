# Firmware

Qui trovi le operazioni “da laboratorio”:

- aggiornare firmware (in modo sicuro)
- capire i preset LoRa (senza leggende metropolitane)
- fare backup/restore configurazioni

---

## Prima di aggiornare (regola d’oro)

1. **Backup configurazione**
2. Annotati:
   - regione
   - modem preset
   - canali e PSK (in modo sicuro)
3. Aggiorna
4. Ripristina configurazione (se necessario)
5. Test base: connessione + messaggi

Vai a: **Config backup/restore**.

---

## Quale firmware scegliere (Italia)

In Italia trovi spesso sia il firmware ufficiale sia quello con patch LoraItalia.

Vai a: **Varianti firmware (Italia)**.
