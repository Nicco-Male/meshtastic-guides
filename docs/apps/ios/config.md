# iOS: configurazioni principali

Le stesse 4 aree “cruciali” valgono anche su iOS:

1. Radio / LoRa (regione, preset)
2. Canali (nome + PSK)
3. Posizione (GPS, intervalli, fixed position)
4. Moduli (telemetria, MQTT, ecc. se supportati)

---

## Suggerimento operativo

- Configura prima radio + canali
- Poi verifica messaggi
- Solo dopo abilita moduli

---
