Most common:
- Bluetooth (scan devices, pair, connect)
- Wi‑Fi/IP (node AP mode or LAN client mode)

If you’re connected but nothing works, double-check:
- correct node
- correct IP (Wi‑Fi)
- region / LoRa preset / channel PSK
