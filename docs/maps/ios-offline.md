# Offline maps su iOS (concetti + note pratiche)

Su iOS la gestione offline dipende dall’app e dal sistema (MapKit/Apple Maps).

Consigli pratici:
- prepara offline *prima* di uscire (in Wi‑Fi)
- verifica che la mappa non sia “grigia” senza dati
- se ti serve un controllo totale sulle tile, valuta workflow con tile downloader e formati compatibili (vedi Tile / download)

!!! note
    Se l’app/firmware supportano mappe custom dipende dalla versione: documentiamo i flussi “robusti” e rimandiamo alle fonti ufficiali per i dettagli.
